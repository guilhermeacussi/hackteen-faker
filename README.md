# Faker - HackTeen 2025 - 1AMS

## Sobre o Projeto

O **Faker** é um projeto desenvolvido para o evento HackTeen 2025, com o objetivo de criar uma aplicação web funcional, interativa e segura.  
O sistema inclui funcionalidades de login, cadastro, painel de usuário e chat em tempo real.

Este projeto foi desenvolvido por **alunos da Etec Bartolomeu Bueno da Silva**, que, no ano do projeto, estavam na turma **1 AMS**.  
O projeto utiliza **PHP, JavaScript, HTML e CSS**, seguindo boas práticas de programação e organização de código.

---

## Funcionalidades

- Cadastro e login de usuários com validação
- Painel (Dashboard) personalizado para cada usuário
- Chat interno para comunicação em tempo real
- Páginas de equipe e informações adicionais
- Mensagens de erro e tratamento de falhas
- Layout responsivo e moderno

---

## Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Banco de Dados:** MySQL
- **Controle de versão:** Git / GitHub

## Como Rodar o Projeto

1. Clone este repositório:

```bash
git clone git@github.com:gacussi/hackteen-2025-1AMS.git
````

2. Entre na pasta do projeto:

```bash
cd hackteen-2025-1AMS
```

3. Configure seu servidor local com suporte a **PHP e MySQL** (XAMPP, WAMP ou similar).

4. Importe o banco de dados `auxilia.sql` (se houver) no MySQL.

5. Abra o navegador e acesse:

```
http://localhost/hackteen-2025-1AMS/index.html
```
